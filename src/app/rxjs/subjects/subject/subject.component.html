A special type of Observable which shares a single execution path among observers
Example
subject: BehaviorSubject<number> = new BehaviorSubject<number>(0);
const sub1 = this.subject.subscribe((data) => console.log(data)) //logs 0,5,10,15
this.subject.next(5);
this.subject.subscribe((data) => console.log(data)).unsubscribe(); //logs 5
this.subject.next(10);
this.subject.subscribe((data) => console.log(data)).unsubscribe(); //logs 10
this.subject.next(15);
sub1.unsubscribe();

Subjects are used for multicasting Observables. This means that Subjects will make sure each subscription gets the exact same value as the Observable execution is shared among the subscribers.
